# AI Experiments

A project built upon a foundation of [AIE Bootstrap - JamesUpdate branch](https://github.com/AcademyOfInteractiveEntertainment/aieBootstrap/tree/james-update) with some minor modifications. The projects intent is to showcase a large variety of Artificial Intelligence behaviours for Game Development.
